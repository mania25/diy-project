\subsection{Detailed Description}
\hyperlink{namespace_apache}{Apache} L\-D\-A\-P library. The A\-P\-R L\-D\-A\-P rebind functions provide an implementation of a rebind procedure that can be used to allow clients to chase referrals, using the same credentials used to log in originally.

Use of this implementation is optional.
