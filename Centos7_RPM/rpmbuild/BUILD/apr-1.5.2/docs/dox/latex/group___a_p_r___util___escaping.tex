Escape control characters in a string, as performed by the shell's 'echo' command. Characters are replaced as follows\-: \textbackslash{}a alert (bell), \textbackslash{}b backspace, \textbackslash{}f form feed, \textbackslash{}n new line, \textbackslash{}r carriage return, \textbackslash{}t horizontal tab, \textbackslash{}v vertical tab, \textbackslash{} backslash.

Any non ascii character will be encoded as '\textbackslash{}x\-H\-H', where H\-H is the hex code of the character.

If quote is not zero, the double quote character will also be escaped.

For efficiency reasons, we don't use escape\-\_\-path\-\_\-segment(), which is provided for reference. Again, R\-F\-C 1808 is where this stuff is defined.
